# **App Name**: LogLens AI

## Core Features:

- Log File Import: Import one or more log files.
- Tabbed Log View: Open each imported log file in a separate tab for easy navigation.
- Log Merging: Merge multiple logs based on timestamps, supporting various timestamp formats.
- Workspace Selection: Switch between Raw, Clustered, and Filtered log views.
- AI Log Chat: Chat with the logs using AI to analyze imported, filtered, or clustered data, powered by ChromaDB as a tool to vectorize logs for conversational capabilities.
- Dashboard Display: Show visual charts and statistics derived from clustered logs, including the most and least frequent log lines, occurrence counts, and anomaly detection.
- Filtering and Highlighting: Apply filters, highlights, and search functionalities within each log view.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke a sense of sophisticated analysis and insight.
- Background color: Dark gray (#303030) to reduce eye strain during long analysis sessions.
- Accent color: Electric green (#7CFC00) for highlighting and call-to-action elements to draw attention.
- Body and headline font: 'Inter', a sans-serif font for a modern, neutral, and easily readable interface.
- Use minimalist and clear icons to represent actions and data types.
- Implement a tabbed interface for log files with a sidebar for navigation and controls.
- Use subtle transitions and animations for workspace changes and data updates.
